# Ingesting Your Knowledge

To get the best answers, you need to feed Axio Hub high-quality data.

## Supported Formats
Currently, we support the following formats:
* **PDF (.pdf):** Best for reports, manuals, and contracts. Text-selectable PDFs work best.
* **Word (.docx):** Great for drafts and internal notes.
* **Text (.txt, .md):** Ideal for raw notes or documentation.

## Google Drive Integration
You can import files directly from Google Drive.
1. Click the **Google Drive** icon in the upload modal.
2. Authenticate securely with your Google account.
3. Select the files you want to sync.

## File Limits
* **Maximum File Size:** 50MB per file.
* **Security:** All files are scanned for malware before processing.
